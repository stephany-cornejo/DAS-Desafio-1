# DesafioDas

A C# WinForms desktop application built with .NET 8.0.

## Project Structure

```
DesafioDas/
├── DesafioDas.csproj       # Project file
├── Program.cs              # Entry point
├── Form1.cs                # Main form logic
└── Form1.Designer.cs       # Form designer file
```

## Getting Started

### Prerequisites

- .NET 8.0 SDK or later
- Visual Studio Code or Visual Studio

### Building the Project

```bash
dotnet build
```

### Running the Project

```bash
dotnet run
```

## Development

The project uses Windows Forms for the UI. The main form is defined in `Form1.cs` and `Form1.Designer.cs`.

To edit the form design:
1. Open `Form1.cs` in Visual Studio or Visual Studio Code
2. Use the form designer to modify the UI
3. Save changes

## Debugging

To debug the application:
1. Open the workspace in Visual Studio Code
2. Press `F5` or use the `Run` menu to start debugging
3. Use breakpoints to step through code
